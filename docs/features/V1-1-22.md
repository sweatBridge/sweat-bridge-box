## 참고 정보

### 아키텍처 패턴
```
AdminUserList (Page)
  ↓
useAdminUsers (Hook)
  ↓
AdminService / UserRepository
  ↓
Firestore
```

### 관련 파일
- `react_web_app/src/pages/admin/AdminUserList.tsx` - 메인 컴포넌트
- `react_web_app/src/pages/admin/_mockData.ts` - 현재 모의 데이터
- `react_web_app/src/types/auth.ts` - UserRole, BoxStatus 타입
- `react_web_app/src/constants/adminColors.ts` - 색상 테마

### Firestore 컬렉션 구조
```
/box/{boxName}
  └── members/
      └── {email}/
          - realName: string
          - nickName: string
          - phone: string
          - role: UserRole (coach | operator | admin)
          - status?: BoxStatus (NONE | PENDING | APPROVED | REJECTED)
          - createdAt: timestamp
```
